# GauPDF Editor — Desktop Application

GauPDF Editor là ứng dụng desktop chỉnh sửa PDF toàn diện, được thiết kế để cạnh tranh với Adobe Acrobat DC. Ứng dụng chạy trực tiếp trên máy tính Windows (Electron framework), cho phép người dùng chỉnh sửa, tạo, xuất, in ấn và gộp file PDF với hiệu năng cao và trải nghiệm native.
